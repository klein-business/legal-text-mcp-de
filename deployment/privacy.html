Privacy

This service is operated by klein.business. It serves the legal-text-mcp-de MCP API.

What we log

  • Request method, path, HTTP status, latency in milliseconds
  • An anonymised bucket derived from the User-Agent header

What we do NOT log

  • Request bodies (MCP tool arguments, prompts, resources)
  • Response bodies
  • Source IP addresses (after rate-limit window expiry)

Data retention

Anonymised log lines are retained for 30 days for operational debugging, then deleted.
